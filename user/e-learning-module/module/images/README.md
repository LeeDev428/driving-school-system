# Images Directory

This directory contains images for the module cards.

## Required Images

For the module cards to display properly, you need to add the following images:

1. **road-signs.jpg** - Image for Road Signs module
2. **road-lines.jpg** - Image for Road Lines module  
3. **regulations.jpg** - Image for Regulations module

## Image Specifications

- **Recommended size**: 300x200 pixels
- **Format**: JPG, PNG, or WebP
- **File size**: Keep under 100KB for optimal loading

## Fallback Images

If the images are not found, the cards will automatically display placeholder images with the module names.

## Example Image URLs

You can use these placeholder URLs for testing:

- Road Signs: `https://via.placeholder.com/300x200/667eea/ffffff?text=Road+Signs`
- Road Lines: `https://via.placeholder.com/300x200/764ba2/ffffff?text=Road+Lines`
- Regulations: `https://via.placeholder.com/300x200/28a745/ffffff?text=Regulations`

## Adding Your Own Images

1. Place your image files in this directory
2. Name them exactly as specified above
3. The images will automatically be displayed in the module cards
